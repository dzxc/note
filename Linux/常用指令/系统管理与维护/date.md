### cd 命令
改变当前工作目录
#### 用法
cd [目录名]

cd [目录] 切换到指定目录下
cd 或 cd ~返回默认工作目录
cd ~[用户名] 返回指定用户工作目录
cd .. 或cd ../ 返回上级工作目录下
cd / 返回根目录下
cd ../..  返回上两级目录；
cd !$  把上个命令的参数作为cd参数使用。

#### 选项
|选项|说明|
|----|----|
|-p| 如果要切换到的目标目录是一个符号连接，直接切换到符号连接指向的目标目录|
|-L |如果要切换的目标目录是一个符号的连接，直接切换到字符连接名代表的目录，而非符号连接所指向的目标目录|
|- |当仅实用"-"一个选项时，当前工作目录将被切换到环境变量"OLDPWD"所表示的目录|
